Welcome to the Snake project!

About the Game

Snake involves the user controlling a moving line (or snake) across an LCD display. The user can move the snake in four possible directions: up, down, left or right. The user must avoid dying which occurs when the snake head makes contact with either of the four walls or any part of its own body. The object of Snake is to eat as many mice as possible. A mouse is represented as a dot or square on the screen and is eaten when the head of the snake makes contact with it. Every time a mouse is eaten:

  1. the snake becomes one unit longer

  2. the player’s score goes up

  3. the snake’s speed increases

  4. a new random mouse is generated on the screen
